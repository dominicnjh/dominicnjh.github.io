Instructions

1.
Press any key to start. Simon will give the first signal. Repeat the signal by pressing the same color button.
2.
Simon will duplicate the 1st signal and add one more. Repeat these two signals by pressing the same color buttons, in order.
3.
Simon will duplicate these 1st two signals and add one.
4.
Continue playing as long as you can repeat each sequence of signals correctly.
